🌍 Deploy on GitHub Pages

Follow these steps carefully:

1. Go to your GitHub repository
2. Click Settings
3. Go to Pages
4. Under Source:
   - Select Deploy from branch
   - Branch → "main"
   - Folder → "/ (root)"
5. Click Save

⏳ Wait 1–2 minutes
Your site will be live at:

https://your-username.github.io/repo-name/

❗ Common Issues & Fixes

1. Site Not Opening (404 Error)

- Ensure file name is exactly: "index.html"
- Must be in root folder (not inside another folder)

---

2. Styles Not Loading

- Check if CSS is inside same file or correctly linked

---

3. JavaScript Not Working

- Ensure "<script>" is added before "</body>"
- No syntax errors in JS

---

4. Blank Page Issue

- Open browser console (F12)
- Check for errors
